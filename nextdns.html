NextDNS Guide

1. Sign Up and Configuration:

- Visit the NextDNS website (nextdns.io) and sign up for an account.

- After signing up, you'll be able to create a custom configuration. This is where you can set filtering options, blocklists, whitelists, and other features.

2. Installing on Desktop Devices:

For Windows:

a. Open Control Panel.

b. Go to Network and Sharing Center > Change adapter settings.

c. Right-click on your active network connection and select 'Properties'.

d. Select 'Internet Protocol Version 4 (TCP/IPv4)' and click on 'Properties'.

e. Choose 'Use the following DNS server addresses' and enter the NextDNS IPv4 addresses provided to you.

f. If you use IPv6, repeat the above steps for 'Internet Protocol Version 6 (TCP/IPv6)'.

For macOS:

a. Go to System Preferences > Network.

b. Select your active network connection and click on 'Advanced'.

c. Go to the DNS tab and add the NextDNS IPv4 (and IPv6 if used) addresses.

Linux:

The steps vary depending on your distribution, but generally:

a. Go to Network settings.

b. Edit your current connection.

c. Set the DNS to manual and add the NextDNS addresses.

3. Installing on Phones:

For Android:

a. Go to Settings > Network & Internet > Wi-Fi.

b. Tap on your connected Wi-Fi network.

c. Tap on 'Advanced' > IP settings > Static.

d. Enter the NextDNS IPv4 (and IPv6 if used) addresses in the DNS fields.

- Note: Android 9+ has a feature called "Private DNS" which can be used to set NextDNS globally for all networks. Look for 'Private DNS' in settings and enter the unique NextDNS hostname provided to you.

For iOS:

a. Go to Settings > Wi-Fi.

b. Tap on the 'i' icon next to your connected network.

c. Under 'DNS', select 'Configure DNS' and choose 'Manual'.

d. Add the NextDNS IPv4 (and IPv6 if used) addresses.

4. Install NextDNS Apps:

- For ease of use, NextDNS offers apps for various platforms. Installing their app can simplify the setup process and may provide enhanced features or stability. Consider downloading their app from the official website or app store.

5. Suggestions:

- After setting up, visit https://my.nextdns.io/start to verify if your device is using NextDNS correctly.

- Remember, when you connect to a new network, you might have to change the DNS settings again unless you're using the NextDNS app.

- Consider using the NextDNS profiles feature for different users or groups of devices.

- Regularly check the NextDNS analytics page to see if there are any unusual or unexpected blocked requests.

Lastly, always ensure that all your clients understand the changes and benefits of using NextDNS, as well as any content filtering settings you've applied. This helps avoid confusion or potential issues down the road.
